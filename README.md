## ml-check

A small script to verify that a pre-selected stack for developing and running various ml-tech is installed.

To run the verification script:

```bash
wget https://github.com/chridahl/ml-setup/blob/master/ml-check.sh
chmod +x ml-check.sh
./ml-check.sh
```
